Prüfe die Dokumentation auf Aktualität:

1. Lies relevante Doku-Dateien (README, AGENTS.md, CLAUDE.md, docs/).
2. Vergleiche mit aktuellem Code (Funktionen, Konfiguration, Setup-Schritte).
3. Befund:
   - **Veraltete Aussagen:** Was stimmt nicht mehr mit dem Code überein?
   - **Fehlende Schritte:** Was fehlt in Setup/Usage-Anleitungen?
   - **Falsche Referenzen:** Falsche Dateinamen, Funktionen, Befehle?
   - **Konkrete Korrekturvorschläge:** Exakter Text der Korrektur, nicht nur "veraltet".

**Verboten:** Keine Codeänderungen ohne separaten Auftrag.
